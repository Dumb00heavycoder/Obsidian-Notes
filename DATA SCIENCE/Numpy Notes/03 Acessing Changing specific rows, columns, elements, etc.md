1)- Accessing elements:- 
	Getting an element:-
		`a[row, column]`
	Getting a row:-
		`a[row:]`
	Getting a Column
		`a[: column]`
	Getting more fancy (start index: end index: stepsize):-
		`a[row index, startindex:endindex: stepsize]`
	3D:-
		Work outside in:-
		`a[dim1, dim2, dim3]`
2)- Changing elements:- 
	A single element:-
		`a[row, column] = 18`
	A Column:-
		`a[: column] = [2,3,5,9]`
	3D:-
		Access With the access method and then change values normally
