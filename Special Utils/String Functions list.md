lower()
	 `print(x.lower())`
upper()
	 `print(x.upper())`
capitalize(): converts first character into capital
	 `print(x.capitalize())`
title(): converts first character of each word upper case
	 `print(x.title())`
swapcase(): will swap the case
	 `print(x.swapcase())`
islower():- returns True if all letters are lower case
	 `print(x.islower())`
isupper() 
	 `print(x.isupper())`
istitle()
	 `print(x.istitle())`
isdigit() 
	 `print(x.isdigit())`
isalpha() 
	 `print(x.isalpha())`
