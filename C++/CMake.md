***Build systems***
Build systems are software tools that automate the complex process of converting source code into executable binaries, managing dependencies, and ensuring consistent, efficient builds.

***CMake***
CMake generates modern buildsystems such as Ninja (build system) as well as project files for IDEs such as Visual Studio and Xcode.
CMake is widely used for the C and C++ languages, but it may be used to build source code of other languages to
